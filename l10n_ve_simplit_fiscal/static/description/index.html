Venezuela - Gestión Fiscal Híbrida (Simplit)

Sistema inteligente de retenciones IVA para Contribuyentes Especiales

Características Principales

✅ Configuración Centralizada

Interfaz intuitiva con pestañas para gestionar toda la configuración fiscal desde un solo lugar.

🤖 Automatización Inteligente

Reemplazo automático de impuestos en facturas de proveedor según las reglas de retención del SENIAT.

💰 Generación Automática

Crea todos los impuestos y grupos necesarios con un solo clic. Incluye:

  • Impuestos base (IVA 16%, IVA 8%)
  • Retenciones (75%, 100%)
  • Grupos de impuestos (Combos)

🔒 Modelo Híbrido

Combina la potencia de Odoo con la seguridad de un API externo:

  • Contabilización inmediata en Odoo
  • Numeración legal diferida vía API
  • Sin saltos ni huecos en la numeración

¿Cómo Funciona?

  1. Configure su empresa como Agente de Retención IVA
  2. Genere los impuestos automáticamente con un clic
  3. Configure sus proveedores con el tipo de retención (75% o 100%)
  4. ¡Listo! Las facturas aplicarán automáticamente las retenciones correctas

Cumplimiento Legal

Diseñado específicamente para cumplir con las normativas del SENIAT en Venezuela. Garantiza el correcto cálculo y registro de retenciones de IVA para Contribuyentes Especiales.
